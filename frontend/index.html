Hustle & Home: Charlotte 30

Manage 30 days of gig work, online selling, and family expenses. Make smart decisions under pressure.

How This Page Works
1) Start Game

Use this for free play. You enter name/city and start an independent 30-day simulation.

2) Join Class Assignment

Use class code to load only that class's active assignments. Choose one assignment and start from teacher settings.

3) Game State -> Daily Allocation -> Daily Log -> Weekly Report

State shows current financial position, Allocation is your daily plan, Log records day-by-day outcomes, Weekly Report summarizes each 7-day block.

4) Sprint Area

A separate 1-hour challenge. You choose one offer per simulated day and get a final performance score vs hidden optimal path.

Quick Start (3 Steps)
Step 1

Register / Sign In with first name, last name, and school email.

Step 2

Enter Class Code, click Join Class, then load and select your active assignment.

Step 3

Start assignment, make daily decisions, then click Turn In Assignment when finished.
