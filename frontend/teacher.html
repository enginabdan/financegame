Teacher Dashboard

Monitor student runs, outcomes, and day-by-day decisions.

How This Page Works
1) Access

Enter API Base URL and Teacher Key, then click Load Dashboard to unlock all teacher tools.

2) Class Setup -> Assignment Setup -> Class Students

Create classes, create assignments under a class, then load roster to set students active/inactive or remove students.

3) Overview, Filters, Sessions, Selected Session Logs

Track all student sessions, filter by class/assignment/status, inspect logs, edit records, or delete incorrect runs.

4) Assignment Rubric + Sprint Leaderboard

Review performance scores/grades for a specific assignment and compare sprint decision quality across students.

5) Trash + Audit Log + Risk Alerts

Restore deleted records, inspect data-action history, and identify students with financial risk patterns.

Quick Start (3 Steps)
Step 1

Paste Teacher Key in Access and click Load Dashboard.

Step 2

Create Class, then create at least one Assignment for that class.

Step 3

Share Class Code with students. Track progress in Sessions and review outcomes in Assignment Rubric.
